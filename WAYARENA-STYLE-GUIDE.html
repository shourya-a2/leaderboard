5. USAGE GUIDELINES

DO

  • ✓ Copy the :root CSS variables block to every new file
  • ✓ Use Press Start 2P for headers and labels only
  • ✓ Use Fredoka for all readable content (names, numbers, descriptions)
  • ✓ Use pixel shadows (hard-edged) for the retro feel
  • ✓ Keep corners sharp (border-radius: 0) for pixel art aesthetic
  • ✓ Use the defined color palette — no custom colors

DON'T

  • ✗ Don't use Inter, Quicksand, Segoe UI, or other fonts
  • ✗ Don't use blur shadows (use pixel shadows instead)
  • ✗ Don't add new colors without updating this guide
  • ✗ Don't mix purple-tinted and brown-tinted backgrounds
  • ✗ Don't use rounded corners for main containers
